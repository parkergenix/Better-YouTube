# Better YouTube

A Safari extension for macOS that cleans up your YouTube experience with a few focused features.

## Features

- **Theater mode** — automatically opens videos in theater mode
- **Auto quality** — sets video quality via a customizable keyboard shortcut
- **Custom shortcut** — bind any key to trigger quality selection on demand

## Requirements

- macOS 13 or later
- Safari

## Installation

1. Download the latest release from the [Releases](https://github.com/parkergenix/Better-YouTube/releases) page
2. Unzip and move **Better YouTube.app** to your Applications folder
3. Open the app — it will prompt you to enable the extension in Safari
4. Go to **Safari → Settings → Extensions** and enable Better YouTube
5. Close the app — the extension runs in Safari from here

## Usage

Click the Better YouTube icon in the Safari toolbar to open the popup and toggle features on or off. The quality shortcut key can be changed by clicking the key cap and pressing any key.

Updates are delivered automatically via the built-in updater. You can also check manually under **Better YouTube → Check for Updates…**
